--------PEM--------
Parallel Email Messanger

PEM is a program made in java that allows the user to sign in with their Gmail account. The program utilizes password and email recognition to access the Client's email account. Then the Client can input an excel sheet of values that correspond to various people and their respective properties. The left column of the excel sheet would be names, while the next column is their respective email. The first row would correspond to fields that the user could input and then under those fields would be a type indicating if that field was a string or a boolean. The program then matches each person to their respective field type, where on the next screen the Client can generate unique messages to be said when it recognize a field type. Therefore, ultimately the user will be able to create a personalized email message for however many people he wanted in just one body. The program will then generate a unique email for each user in the excel sheet.
